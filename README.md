# Cloned site

Generated by **clonner** from `https://matter-test.my.canva.site/copy-of-coach-aor-ca-website-design-laptop-version`.

```bash
npm install
npm run dev
```

Then open http://localhost:3000.

## What's here

- `app/clone.html` — the captured, fully-rendered page markup (asset URLs localised).
- `app/clone.css` — every stylesheet from the original, concatenated in cascade order, fonts/images localised.
- `public/assets/` — all downloaded images and fonts.
- Tailwind is installed and ready (preflight off so it won't disturb the clone) for when you start refactoring sections into real components.

## Refactoring toward a real site

The clone is a faithful 1:1 snapshot. To evolve it:
1. Identify a section in `clone.html`.
2. Rebuild it as a Tailwind component under `app/`.
3. Remove the matching markup from `clone.html`.
Repeat section by section until the static snapshot is gone.
